## Getting started

```npm i```

```docker compose up```

```npm run dev```

## You need to create .env file

```
ENV_REPOSITORIES=mongo
SECRET_TOKEN=YOUR TOKEN
URI_MONGO=YOUR MONGO DB
SECRET_TOKEN_BUILDER=YOUR SECRET
SECRET_TOKEN_HEDERA_APP=YOUR SECRET
PORT=9000
```

You need  docker, a stripe (dev) account and your hedera account keys.
