## Getting started

```npm i```

```docker compose up```

```npm run dev```

## You need to create .env file

```
PORT=3012
SECRET_TOKEN_COMPANY=
SECRET_TOKEN_EMPLOYEE=
SECRET_TOKEN_ADMIN=
ENV_REPOSITORIES=mongo
URI_MONGO=mongodb://localhost:27026/arze
URI_BACK=http://localhost:3012/
ADMIN_API_KEY=
CONTRACT_SECRET_KEY=
CONTRACT_ALGORITHM=aes-256-cbc
CONTRACT_IV_LENGTH=16
HEDERA_ACCOUNT_ID=
HEDERA_PRIVATE_KEY=
STRIPE_SECRET_KEY=
```

You need  docker, a stripe (dev) account and your hedera account keys.
